# Azure QuickTask

A simple cloud-based To-Do Task Manager built during Microsoft Elevate Azure Fundamentals Internship.  

## How to Run Locally
1. Clone repo: `git clone https://github.com/YourUsername/Azure-QuickTask.git`  
2. Navigate: `cd Azure-QuickTask`  
3. Activate virtual environment: `venv\Scripts\activate`  
4. Install dependencies: `pip install -r requirements.txt`  
5. Run app: `python app.py`  
6. Open browser: `http://127.0.0.1:5000`
